Uma classe Node:  
```  
class Node:  
    def __init__(self,initdata):  
        self.data = initdata  
        self.next = None  
  
    def getData(self):  
        return self.data  
  
    def getNext(self):  
        return self.next  
  
    def setData(self,newdata):  
        self.data = newdata  
  
    def setNext(self,newnext):  
        self.next = newnext  
```  
  
Criamos um objeto Node da maneira usual.  
  
>>> temp = Node(93)  
>>> temp.getData()  
93
